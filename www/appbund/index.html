< The appbund script

appbund

The appbund script creates a file that can be used to start ArgoUML from on a Mac. See Technical Note TN2031 for details on how this format works.

Tools used

It must be run on a Cygwin system (probably a Unix system will do).

On top of the really basic unix commands (sh, mkdir, ls, cp, cat, rm) the following are needed: sed, awk, tar, and gzip.
