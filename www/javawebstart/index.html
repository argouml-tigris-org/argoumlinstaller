javawebstart

The script is called createJWS.sh.

The javawebstart script copies the files to the maven2-like structure in the argouml-downloads project used for storing the javawebstart jars and creates the jnlp file.

The version of each jar file is controlled by the LAYOUT file. A file in the LAYOUT file is either "argouml-provided" i.e. it will be created with the same version as the argouml release or not "argouml-provided" i.e. it will be created with the version number in the LAYOUT file that is the same version number as that jar file is known with.

Tools used

It must be run on a Cygwin system (probably a Unix system will do).

On top of the really basic unix commands (sh, mkdir, ls, cp, cat, rm, test, echo, basename, tr, chmod) the following are needed: grep, sed, awk, cmp, unzip, diff
