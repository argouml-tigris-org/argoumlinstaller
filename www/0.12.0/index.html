About ArgoUML

ArgoUML is an object oriented design tool with special user interface features that make it easier to learn UML and to make good designs. ArgoUML is open source and available free of charge.

ArgoUML is a great choice for people who want to use the Unified Modeling Langage:

  • In education, industry or government;
  • Without the expense and delay of a purchase cycle;
  • Through a pleasant and efficent user interface;
  • In teams where many people need to access UML designs;
  • When compliance to the UML standard is very important; or,
  • When common design errors cannot be tolerated.

System requirements

CPU Speed:350MHz or greater
System RAM:128MB or greater
Disk Space:6MB (ArgoUML) + 20MB (JVM, if being installed)
Operating System:Windows 98/NT/2000/XP, Linux, Solaris, or other with Java

Install ArgoUML with JavaWebStart

JavaWebStart can install and upgrade ArgoUML over the web using the latest files available. Once the application been run twice, it can be launched from the desktop or Start menu, without an internet connection.

This is your best choice if you have a fast internet connection.

To use JavaWebStart, see the instructions on ArgoUML.org

Install ArgoUML as an Application

Installing ArgoUML as an application will allow you to launch it from the Start menu. If you ever need to upgrade the software, you will need a new installer for the new version.

This is your best choice if you do not have a fast internet connection.

To install ArgoUML as an application, click here.
