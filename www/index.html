Mission

This project contains installers for ArgoUML.

There is one directory for each installer. Each directory contains the configuration files, and tools, needed to build a package using that installer.

The directory contains a script doit.bat if that installer is available on a windows platform, and doit.sh if that installer is available on a unix platform. For convenience the script doall.bat runs all the installers that works on a windows platform, and doall.sh runs all the installers that works on a unix platform. All scripts assume that argouml has been built or downloaded and available in ../argouml/build.

The documentation of the specifics for each installer is in a directory in the web site part of this directory. Each such set contains:

  • names and references of the tools needed
  • information on the platform needed to run that installer
  • the list of tools to install and links to installation instructions.

The result of running the installers is either published somewhere on the web (the argouml-downloads project is one possibility) or available only for the runner (for burning on cd:s). (This project was previously used for the purpose of publishing releases. This is not a good idea and will be discontinued.)
