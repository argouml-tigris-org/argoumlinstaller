nsis

The nsis script was created by Dave Thompson.

It creates a windows installer using the Nullsoft Scriptable Installer System (NSIS). The resulting installer is a single .exe file, and runs on Windows only, doing following:

  • Checks for a suitable Java Runtime Environment on the target machine, and offers to download and install one if necessary.
  • Copies the ArgoUML .jar files to a user specfied install directory (default 'C:\Program Files\ArgoUML').
  • Creates an icon on the desktop to launch ArgoUML.
  • Creates an uninstaller.
  • Adds entries to windows registry to allow uninstallation via Add/Remove Programs.
