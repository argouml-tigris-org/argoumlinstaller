Libraries used by the IzPack installers for ArgoUML and modules:
* BerliOS IzPack: izpack-standalone-compiler-3.8.1 is the file standalone-compiler.jar of the release 3.8.1 of the IzPack library. You can fetch the library at: http://www.izforge.com/izpack/
* Ant 1.6.5: ant-1.6.5.jar and ant-launcher-1.6.5.jar are the files from the release 1.6.5 of Apache Ant. You can fetch the library at: http://ant.apache.org
